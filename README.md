# graqphl-fileupload-websocket-full-stack-app

## Run Front End Code

* Go inside frontend-ui
* npm install
* npm run

open localhost:3000 in browser


## Run Start Backend End Code

* Go inside backend folder
* gradle build
* gradle bootRun
